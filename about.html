About project

PolishStocksEye is a data visualization project of Warsaw Stock Exchange data using Power BI, PostgreSQL, Docker, Python.

The project aims to enable quick analysis of companies listed on the Warsaw Stock Exchange.
Due to the process of connecting the Power BI gateway to a local PostgreSQL database, the data is not currently updated in real-time.
This is a non-profit project. The author does not guarantee the accuracy, completeness, or timeliness of the data. The content provided is for informational purposes only and does not constitute investment advice.
