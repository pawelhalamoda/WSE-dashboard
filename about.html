O projekcie

PolishStocksEye is a data visualization project of Warsaw Stock Exchange data using Power BI, PostgreSQL, Docker, Python. The project aims to enable quick analysis of companies listed on the Warsaw Stock Exchange. Due to the process of connecting the Power BI gateway to a local PostgreSQL database, the data is not currently updated in real-time. The project is authored by Paweł Halamoda. LinkedIn profile
